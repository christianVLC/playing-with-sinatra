# playing-with-sinatra

Puppet Modules for install Forecast App with Sinatra for show the temperature of Valencia.

- Puppet Server: Puppet4 + Hiera:
 Classes/Modules are in the profiles included and profiles are in the role (Hiera) included.

- Puppet Server + App Server are Debian Jessie

- Modules are tested with Docker.
